Do your Maths









Instructions

  1. Click on the button with the operation that you would like to do.
  2. There will be 2 random number placed with the operation that you selected.
  3. type the answer in the text input box given.
  4. Once you have typed the answer click check and if your answer is correct then a new question will appear but if it is wrong you have to solve it again.
  5. if you want to go back and do another operation click on the back button.
